vaje: FOR, WHILE, DO-WHILE

Stavek for, while, do-while

Zapisali bomo program v programskem jeziku java, ki prikazuje uporabo zank.

public class ForWhileDoWile {
public static void main(String[] args) {

for (int stevecFor = 1; stevecFor <= 10; stevecFor++) {
System.out.println("VrsticaFor " + stevecFor);
}

System.out.println();

int stevecWhile = 1;
while (stevecWhile <= 10) {
System.out.println("VrsticaWhile " + stevecWhile);
stevecWhile++;
}

System.out.println();

int stevecDoWhile = 1;
do {
System.out.println("VrsticaDoWhile " + stevecDoWhile);
stevecDoWhile++;
} while (stevecDoWhile <= 10);
}
}
