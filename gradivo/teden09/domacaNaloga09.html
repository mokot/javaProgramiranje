Domača naloga: IF, IF-ELSE, IF-ELSEIF-ELSE

1. Naloga: Napišite program v programskem jeziku java, ki prebere celo število kilometrov in jih pretvori v milje.
(1 kilometer znaša 0.621371192 milje.)
Primer: program prebere število 100 in izpiše "62.1"
2. Naloga: Napišite program v programskem jeziku java, ki prebere temperaturo v stopinjah celzij in izpiše temperaturo v stopinjah farenhajtov.
(Pretvorba: celzij * 1.8 = fahrenheit - 32)
Primer: program prebere število 32.0 in izpiše "89.6"
3. Naloga: Napišite program v programskem jeziku java, ki prebere celo število in za to število napiše "sodo", če je število sodo, oziroma "liho", če je število liho.
Primer: program prebere število 42 in izpiše "sodo"
4. Naloga: Napišite program v programskem jeziku java, ki prebere črko in izpiše "samoglasnik", če je prebran znak samoglasnik, oziroma "soglasnik", če je znak soglasnik.
Primer: program prebere število 'a' in izpiše "samoglasnik"
5. Naloga: Napišite program v programskem jeziku java, ki znak prebere in izpiše "DA", če je prebran znak črka, oziroma "NE", če prebran znak ni črka.
Primer: program prebere znak '%' in izpiše "NE"
6. Naloga: Napišite program v programskem jeziku java, ki prebere tri cela števila in izpiše najmanjše število izmed njih.
Primer: program prebere števila 8, 19, -42 in izpiše "-42"
* 7. Naloga: Napišite program v programskem jeziku java, ki prebere realna števila a, b in c ter reši sistem kvadratne enačbe.
(Kvadratna enačba je v realnem prostoru rešljiva natanko takrat, kadar je diskriminanta večja od 0. Če je diskriminanta strogo večja od nič, ima kvadratna enačba dve različni rešitvi, če pa je diskriminanta enaka 0, ima enačba eno dvojno rešitev.)
Primer: program prebere števila 2.0, 4.0, -4.0 in izpiše "x1 = 0.73, x2 = -2.73"
