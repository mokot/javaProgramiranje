Stavek switch

Zapisali bomo program v programskem jeziku java, ki prikazuje uporabo stavka switch.

import java.util.Scanner;

public class Switch {
public static void main(String[] args) {

Scanner sc - new Scanner(System.in);

int ocena = sc.nextInt();

switch (ocena) {
case 5:
System.out.println("odlicno");
break;
case 4:
System.out.println("prav dobro");
break;
case 3:
System.out.println("dobro");
break;
case 2:
System.out.println("zadostno");
break;
case 1:
System.out.println("nezadostno");
break;
default:
System.out.println("neveljavna ocena");
break;
}
}
}

Pogojni operator

Zapisali bomo program v programskem jeziku java, ki prikazuje uporabo pogojnega operatorja.

public class PogojniOperator {
public static void main(String[] args) {

int stevilo = 42;
String resitev;

resitev = (stevilo == 42) ? ("odgovor obstaja") : ("navadno stevilo");

System.out.println(resitev);
}
}

Stavek break

Zapisali bomo program v programskem jeziku java, ki prikazuje uporabo stavka break.

public class Break {
public static void main(String[] args) {

for (int stevilo = 1; stevilo <= 10; stevilo++) {

if (stevilo == 5) {
break;
}

System.out.println(stevilo);
}
}

Stavek continue

Zapisali bomo program v programskem jeziku java, ki prikazuje uporabo stavka continue.

public class Continue {
public static void main(String[] args) {

for (int stevilo = 1; stevilo <= 10; stevilo++) {

if (stevilo >= 5 || stevilo <= 9) {
continue;
}

System.out.println(stevilo);
}
}
