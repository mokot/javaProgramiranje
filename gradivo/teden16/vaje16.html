Indeksi in elementi

Zapisali bomo program v programskem jeziku java, ki prikazuje dostop do elementov s pomočjo indeksov za sledečo tabelo:

Indeks: 0 1 2 3 4 5
Element: 25 42 17 54 20 36


public class IndeksiElementi {
public static void main(String[] args) {
int[] tabela = {25, 42, 17, 54, 20, 36};

System.out.println(tabela[0]);
System.out.println(tabela[3]);

tabela[3] = -9;
System.out.println(tabela[3]);

System.out.println(tabela.length);
System.out.println(tabela[tabela.length - 1]);

System.out.println(tabela[6]);
System.out.println(tabela[-1]);
}
}

Program nam izpiše:

25
54
-9
6
36
ArrayIndexOutOfBoundsException
ArrayIndexOutOfBoundsException

Izdelava tabele

Zapisali bomo program v programskem jeziku java, ki prikazuje izdelavo tabel s seznamom elementov in s privzetimi vrednostmi elementov.

public class IzdelavaTabele {

public static void main(String[] args) {
int[] popolna = {6, 28, 496, 8128};

boolean[] metiKovanca = new boolean[100];

String[] odgovori = {"DA", "NE", "MORDA"};

char[] samoglasniki = {'a', 'e', 'i', 'o', 'u'};

float[] rezultati = new float[3];

double[] dolzinaNic = new double[0];
}
}

Tabele na koncu programa:

popolna: [6, 28, 496, 8128]
metiKovanca: [false, false, ... , false]
odgovori: ["DA", "NE", "MORDA"]
samoglasniki: ['a', 'e', 'i', 'o', 'u']
rezultati: [0.0f, 0.0f, 0.0f]
dolzinaNic: []
