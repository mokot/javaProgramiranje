Domača naloga: OPERATORJI_1

1. Naloga: Napišite program v programskem jeziku java, ki nad dvema realnima številoma izvede vse naučene aritmetične operacije.
2. Naloga: Napišite program v programskem jeziku java, ki nad dvema realnima številoma izvede vse naučene primerjalne operacije.
3. Naloga: Napišite program v programskem jeziku java, ki nad dvema logičnima izrazoma izvede vse naučene logične operacije.
4. Naloga: Narišite diagram poteka za algoritem, ki s pomočjo naučenega aritmetičnega operatorja * izvede potenciranje nad prebranima celima številoma x in y.
Primer: število x je enako 2, število y pa 7, program izpiše "128"
5. Naloga: Narišite diagram poteka za algoritem, ki izračuna ostanek pri deljenju prebranega celega števila n s prebranim celim številom a brez uporabe aritmetičnega operatorja %.
Primer: število n je enako 17, število a je enako 3, program izpiše "2"
