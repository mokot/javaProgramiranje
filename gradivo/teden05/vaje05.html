Določitev vrednosti

Zapisali bomo program v programskem jeziku java, ki določi vrednosti spremenljivkam.

public class DolocitevVrednosti {
public static void main(String[] args) {

int stevilo1, stevilo2, stevilo3;
stevilo1 = 42;

stevilo2 = stevilo1;
System.out.println("stevilo1: " + stevilo1);
System.out.println("stevilo2: " + stevilo2);

stevilo3 = stevilo2;
System.out.println("stevilo2: " + stevilo2);
System.out.println("stevilo3: " + stevilo3);
}
}

Program nam izpiše:

stevilo1: 42
stevilo2: 42
stevilo2: 42
stevilo3: 42

Aritmetični operatorji

public class AritmeticniOperatorji {

double stevilo1 = 12.4, stevilo2 = 3.6, rezultat;

rezultat = stevilo1 + stevilo2;
System.out.println("stevilo1 + stevilo2 = " + rezultat);

rezultat = stevilo1 - stevilo2;
System.out.println("stevilo1 - stevilo2 = " + rezultat);

rezultat = stevilo1 * stevilo2;
System.out.println("stevilo1 * stevilo2 = " + rezultat);

rezultat = stevilo1 / stevilo2;
System.out.println("stevilo1 / stevilo2 = " + rezultat);

rezultat = stevilo1 % stevilo2;
System.out.println("stevilo1 % stevilo2 = " + rezultat);

String beseda1 = "Programiranje ";

String beseda3 = "zabavno!";
String rezultatNiz = beseda1 + beseda2 + beseda3;

}
}

Enojni operatorji

Zapisali bomo program v programskem jeziku java, ki prikazuje uporabo enojnih operatorjev.

public class EnojniOperatorji {
public static void main(String[] args) {

int stevilo = 42;

System.out.println("+stevilo = " + +stevilo);

System.out.println("-stevilo = " + -stevilo);

System.out.println("++stevilo = " + ++stevilo);

System.out.println("--stevilo = " + --stevilo);
}
}

Program nam izpiše:

+stevilo = 42
-stevilo = -42
++stevilo = 43
--stevilo = 42

Logični operatorji

Zapisali bomo program v programskem jeziku java, ki prikazuje uporabo logičnih operatorjev.

public class LogicniOperatorji {
public static void main(String[] args) {

int stevilo1 = 1, stevilo2 = 2, stevilo3 = 9;
boolean rezultat;

rezultat = (stevilo1 > stevilo2) && (stevilo3 > stevilo1); System.out.println(rezultat);

rezultat = (stevilo1 > stevilo2) || (stevilo3 > stevilo1); System.out.println(rezultat);

rezultat = !rezultat;
System.out.println(rezultat);
}
}

Program nam izpiše:

false
true
false
