Metode standardnih knjižnic

Zapisali bomo program v programskem jeziku java, ki prikazuje uporabo metod iz standardnih knjižnic.

public class StandardneMetode {
public static void main(String[] args) {

System.out.println("Koren 16 je: " + (int) Math.sqrt(16));
}
}

Program nam izpiše:

Koren 16 je: 4

Uporabnikova metoda

Zapisali bomo program v programskem jeziku java, ki prikazuje definicijo in klic uporabnikove metode.

public class UporabnikovaMetoda {

public static void main(String[] args) {
System.out.println("Smo v metodi main");

mojaMetoda();

System.out.println("Smo v metodi main");
}

public static void mojaMetoda() {
System.out.println("Smo v metodi mojaMetoda");
}

}

Program nam izpiše:

Smo v metodi main
Smo v metodi mojaMetoda
Smo v metodi main

Prenos parametra

Zapisali bomo program v programskem jeziku java, ki prikazuje prenos parametra metodi.

public class PrenosParametra {

public static void main(String[] args) {

int stevilo = 42;

mojaMetoda(stevilo);
System.out.println(stevilo);
}

public static void mojaMetoda(int stevilo) {
stevilo++;
System.out.println(stevilo);
}

}

Program nam izpiše:

43
42
