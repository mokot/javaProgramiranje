Prenos parametrov

Zapisali bomo program v programskem jeziku java, ki prikazuje prenos parametrov metodi. Zapisana metoda n-krat izpiše podani znak in po želji doda še prelom vrstice. Parametri: n je prelom vrstice, znak je znak, ki se naj izpiše, prelom pa je parameter, ki v primeru, da ima vrednost true, naredi prelom vrstice.

public class PrenosParametrov {

public static void main(String[] args) {
int n = 42;
char znak = '+';
boolean prelom = true;

zaporedje(n, znak, prelom);
}

public static void zaporedje(int n, char znak, boolean prelom) {
for (int i = 1; i <= n; i++) {
System.out.print(znak);
}

if (prelom) {
System.out.println();
}
}

}

Vračanje vrednosti tipa void

Zapisali bomo program v programskem jeziku java, ki prikazuje prenos parametrov metodi in vračanje vrednosti. Zapisana metoda n-krat izpiše podani znak in po želji doda še prelom vrstice. V primeru, da ja parameter n manjši ali enak 0 se metoda predčasno zaključi. Parametri: n je prelom vrstice, znak je znak, ki se naj izpiše, prelom pa je parameter, ki v primeru, da ima vrednost true, naredi prelom vrstice.

public class VracanjeVrednosti {

public static void main(String[] args) {
int n = 42;
char znak = '+';
boolean prelom = true;

zaporedje(n, znak, prelom);
}

public static void zaporedje(int n, char znak, boolean prelom) {

if (n <= 0) {
return; // takoj zakljuci metodo
}

for (int i = 1; i <= n; i++) {
System.out.print(znak);
}

if (prelom) {
System.out.println();
}
}

}

public static void main(String[] args) {
int stranica1 = 42;
int stranica2 = 24;

int obseg = obsegPravokotnika(stranica1, stranica2);
System.out.println(obseg);

System.out.println(3 * obsegPravokotnika(7, 2) - 6);

int p = 10;
int q = 20;
if (obsegPravokotnika(p, q) == (obsegPravokotnika(p - 1, q + 1)) {
System.out.println("Obsega sta enaka");

}

public static int obsegPravokotnika(int a, int b) {
