vaje: PRVI PROGRAM

Prvi program:

Zapisali bomo naš prvi program v programskem jeziku java. Naloga programa je, da izpiše "Hello, World!".

V poljubnem tekstovnem urejevalniku naredimo novo datoteko imenovano HelloWorld.java.

public class HelloWorld {
public static void main(String[] args) {
System.out.println("Hello, World!");
}
}

Ko imamo naš prvi program napisan, ga moramo najprej pretvoriti v obliko, ki jo lahko nato izvede JVM. To naredimo tako: javac HelloWorld.java. Tako dobimo datoteko imenovano HelloWorld.class. Sedaj moramo program še pognati, to storimo tako: java HelloWorld. Ko program poženemo, nam ta izpiše:

Hello, World!

Napačen program:

Ponovno imamo željo zapisati program, ki izpiše "Hello, World!", ampak smo se pri samem pisanju programa zmotili.

V poljubnem tekstovnem urejevalniku naredimo novo datoteko imenovano HelloWorldNapaka.java
in zapišemo sledečo kodo.

public class HelloWorldNapaka {
public static void main(String[] args) {
system.out.println("Hello, World!");
}
}

Ponovimo postopek pretvorbe programa v obliko ustrezno JVM (HelloWorldNapaka.java). Pri pretvorbi pride do napake (sintaktične). Prevajalnik nam izpiše:

HelloWorldNapaka.java:3: error: package system does not exist
system.out.println("Hello, World!");
^
1 error

Iz zgornjega zapisa lahko natančno zvemo, kje v našem programu se napaka nahaja. Napaka se nahaja v programu HelloWorldNapaka.java, in sicer v 3. vrstici. Ugotovimo lahko tudi, koliko je vseh napak, v našem primeru je samo 1.
