Domača naloga: PRVI PROGRAM

1. Naloga: Napišite program v programskem jeziku java, ki izpiše "Učim se programirati.".
Primer: program izpiše "Učim se programirati."
2. Naloga: Napišite program v programskem jeziku java, ki izračuna in izpiše vrednost sledečega izraza:
"(33 + 5 * 6) / 7".
Primer: program izpiše "9"
3. Naloga: Napišite program v programskem jeziku java, ki izpiše vaše ime in priimek ter leto starosti v formatu: "ime priimek (leto)".
Primer: program izpiše "Rok Mokotar 2000"
* 4. Naloga: Narišite diagram poteka za algoritem, ki ugotovi, če za vnešeno celo število obstaja prijateljsko število. (V matematiki sta celi števili prijateljski, kadar je vsota njunih pravih deliteljev križno enaka drugemu številu. Na primer, števili 220 in 284 sta prijateljski, saj je vsota pravih deliteljev števila 220 {1, 2, 4, 5, 10, 11, 20, 22, 44, 55, 110} enaka 284, vsota pravih deliteljev števila 284 {1, 2, 4, 71, 142} pa je enaka 220.)
* 5. Naloga: Narišite diagram poteka za algoritem, ki izpiše prvih n členov Fibonaccijevega zaporedja. (Fibonaccijeva števila, ki določajo Fibonaccijevo zaporedje, so v matematiki določena kot vsota prejšnjih dveh členov zaporedja. Prvi in drugi člen zaporedja je enak ena, tretji člen je enak vsoti prvega in drugega (1 + 1 = 2), četrti člen je enak vsoti drugega in tretjega (1 + 2 = 3), itd.)
